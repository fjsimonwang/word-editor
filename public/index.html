Click any button → the SDK sends a postMessage command to the embedded editor.

📄 .docx import/export

Open and save native Word documents without any server-side conversion. Round-trips formatting, comments, and page setup.

👥 Real-time collaboration

WebSocket-based presence and document sync. Multiple users can edit the same document simultaneously.

📝 Track changes

Full revision tracking — insertions and deletions are recorded. Accept or reject individual changes.

💬 Comments

Anchor comments to specific text selections. Reply threads, resolve, and delete. Persisted across sessions.

📐 Page layout

Margins, page size (Letter / A4 / Legal / A3), orientation, headers, footers, and page numbers.

📚 Version history

Automatic snapshots on every save. Browse and restore any previous version with one click.

🔍 Find & replace

Search document text with optional regex and case matching. Replace individual or all occurrences.

🎨 Rich formatting

Bold, italic, underline, strikethrough, subscript, superscript, font family, size, color, highlight, alignment, indent.

🖼 Images, tables, shapes

Insert inline images, WordArt, shapes, tables, hyperlinks, symbols, page breaks, horizontal rules.

🔌 Embed SDK

Mount the editor in any page with <script src="sdk.js">. Full postMessage API — 50+ commands.

🌐 REST API

All document operations available as HTTP endpoints. Use the WordEditorRestClient from any language.

⚡ Single binary

Zero-dependency Node.js server. No npm install, no database — just node server.js and it runs.
