Embed WordEditor via iframe + JS SDK

The WordEditor SDK creates an iframe hosting the editor in another origin (or same-origin) and exposes every editor action over window.postMessage. Include <script src="http://editor-host/js/sdk.js"></script> then call WordEditor.init().

// ─── bootstrap ─── const ed = WordEditor.init({ container: "#editor-holder", // CSS selector or Element baseUrl: "http://localhost:3001", user: "Alice", onReady(info) { console.log("ready", info); }, onChange(info) { console.log("change", info); }, }); // ─── all SDK methods return Promises ─── const meta = await ed.getMeta(); const html = (await ed.getContent()).html; await ed.setContent("<p>Hello</p>"); await ed.bold(); await ed.justifyCenter(); await ed.insertImage("https://picsum.photos/360/200", { width:360, height:200 }); await ed.exportDoc("docx"); await ed.setPageSetup({ size:"A4", margins: { top:1, left:1 } }); const docs = await ed.listDocuments(); await ed.newDocument("Memo"); await ed.addComment("Review this paragraph"); ed.destroy();

WordEditorRestClient — use the full API over HTTP

WordEditorRestClient mirrors the embed SDK's method names but talks to the server directly via REST API calls. No iframe, no postMessage — any HTTP client (curl, Python, Postman) can use these same endpoints.

// Browser usage const api = new WordEditorRestClient("http://localhost:3001"); // Document management await api.createDocument("My Document"); await api.loadDocument(docId); const list = await api.listDocuments(); await api.deleteDocument(docId); // Content const content = await api.getContent(); // {html} await api.setContent("<p>Hello</p>"); await api.insertHtml("<p>More</p>"); await api.insertText("plain text"); const meta = await api.getMeta(); // {id, title, rev, words, chars, ...} await api.setTitle("New Title"); // Formatting by HTML fragment const bold = await api.formatHtml("bold", "text to embolden"); // {ok, html: "<strong>...</strong>"} await api.bold("fragment"); await api.justifyCenter("<p>centered</p>"); // Page setup, headers, comments await api.setPageSetup({ size: "A4", margins: { top: 1, bottom: 1 } }); await api.setHeader("Chapter 1", "right"); await api.addComment("Looks good!", "Alice"); // Export await api.exportDoc("docx"); // downloads await api.exportDoc("html"); // Versions const versions = await api.listVersions(); await api.restoreVersion(0);
